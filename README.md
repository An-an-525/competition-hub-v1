# 🏆 学科竞赛智能助手 - Competition Hub

> 神的起源-竞赛系统 | 全国高校大学生竞赛信息查询平台

## ✨ 功能特性

- **🤖 AI 智能问答** — 离线知识库 + 在线 AI 双引擎，即时回答学业、竞赛、校园相关问题
- **📚 学业助手** — 67个专业详情（课程/就业方向）、GPA计算器、考试安排
- **🏫 校园生活** — 校区信息、食堂、宿舍、图书馆、校车路线、常用链接
- **📊 招生咨询** — 录取分数线、学费标准、奖学金、联系方式
- **🏆 学科竞赛** — 55项全国性竞赛信息，含竞赛日历、参赛指南、战绩追踪
- **🔧 工具箱** — 番茄钟、倒计时、课程表、GPA计算器
- **🔍 全局搜索** — Cmd+K 快捷搜索，覆盖所有数据
- **🌙 深色模式** — 自动跟随系统主题

## 🚀 快速开始

直接打开 `index.html` 即可使用，无需安装任何依赖。

也可以在线访问：[GitHub Pages](https://an-an-525.github.io/competition-hub/)

## 📋 技术栈

- 纯 HTML + CSS + JavaScript（单文件应用）
- 无框架依赖，零构建步骤
- localStorage 本地数据持久化
- 响应式设计，支持桌面端和移动端

